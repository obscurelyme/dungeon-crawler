# Dungeon Crawler

A simple dungeon-crawler game made with pygame

> [!NOTE]
> This is a sample game that is being made as a result of a Udemy course. > There are no plans to make this a full fledged game at this time.

Install deps

```sh
uv sync
```

Run the project

```sh
uv run main.py
```

Build the project into a standalone executable

```sh
uv run pyinstaller --onefile --window main.py
```

Run the exe

```sh
./dist/main.exe
```
